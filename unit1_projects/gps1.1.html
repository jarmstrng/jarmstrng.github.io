Pair Programming and DBC

Brandon and Jessica's First Session

What is it?

It is an opportunity for two developers to work collaboratively on the same piece of code, usually using one computer and two screens. This is a common practice in many large tech companies.

Why is it impoortant to Dev Bootcamp?

The most compelling argument for pairing in a learning environment is that you learn better by teaching. Pairing is a great opportunity to both learn and teach.

Defining Driver and Navigator

Driver

The driver types the code. They are in charge of the keyboard and mouse and ensure the code is written properly.

Navigator

The navigator tells the driver what to type.
